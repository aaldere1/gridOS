# gridOS 1.0.8

gridOS 1.0.8 is a terminal polish patch on top of the HUD and provider
expansion release. It tightens the core pane experience so typing, resizing,
copying, and pasting feel predictable in the part of the app people touch all
day.

## What's New

- Reduced terminal rendering jitter while typing and resizing panes.
- Made terminal surface updates idempotent so SwiftUI refreshes do not churn the
  underlying terminal view unnecessarily.
- Restored Command-V paste behavior inside the focused terminal pane.
- Added terminal-aware Command-A and Command-C handling for selecting and
  copying terminal output to the macOS pasteboard.
- Kept the standard Edit menu behavior intact for normal text fields and
  settings controls.
- Added active-pane Terminal menu actions for select all, copy, paste, clear,
  and reset flows.

## Why It Stands Out

The app already looked sharper in 1.0.7. This patch makes it feel calmer and
more trustworthy under your hands. Terminal text should no longer shimmy during
ordinary typing or window changes, and the expected Mac clipboard shortcuts now
work where users expect them: in the focused terminal.

## Install

1. Download `gridOS-1.0.8-16-c60fecb.dmg` from the `v1.0.8` GitHub release.
2. Open the DMG.
3. Drag `gridOS.app` into Applications.
4. If you installed an earlier copy, replace it.
5. Launch gridOS from Applications.

If macOS blocks the app, do not bypass Gatekeeper. Report it as a release issue.
