# gridOS 1.0.5

gridOS 1.0.5 focuses on making AI Command Helper understandable the first time
someone presses Command-K, and on making Settings behave like a real macOS
window.

## What's New

- AI Command Helper now explains each mode directly in the Command-K panel:
  when to use it, what the user provides, and what comes back.
- Command-K setup states are clearer: missing provider keys are called out
  without implying anything leaves the app.
- The helper now uses concrete examples for suggest, explain, and fix flows.
- AI Command Helper settings include a compact "How it works" card covering
  Command-K, mode choice, preview, and provider setup.
- Settings now opens in a custom resizable macOS window.
- The visible app version now derives from bundle metadata, so the UI cannot
  drift from the signed release version.

## Why It Stands Out

The standout feature is still the local-first command cockpit: multi-pane native
terminal work, visual system signal, and an AI helper that treats shell commands
as reviewed actions instead of magic text to run blindly. This release makes
that promise obvious in the product instead of requiring explanation.

## Install

1. Download `gridOS-1.0.5-13-379289a.dmg`.
2. Open the DMG.
3. Drag `gridOS.app` into Applications.
4. If you installed an earlier copy, replace it.
5. Launch gridOS from Applications.

If macOS blocks the app, do not bypass Gatekeeper. Report it as a release issue.
