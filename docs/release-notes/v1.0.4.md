# gridOS 1.0.4

gridOS is a local-first Mac terminal built around a distinctive visual
workspace, multi-pane flow, and guarded AI help for commands you want to
understand before you run.

## What's New

- AI Command Helper now supports both Anthropic and OpenAI API keys.
- Model defaults are refreshed for current OpenAI and Anthropic options.
- Command-K gives clearer setup states, more realistic examples, and safer
  preview-first command handling.
- Open Folder creates a terminal pane directly inside a project directory.
- Live shell sessions now ask before closing a pane, window, or the whole app.
- Restored four-pane layouts clip cleanly instead of letting terminal text bleed
  across panes.
- The DMG now has a polished drag-to-Applications layout with a visible
  Applications shortcut and signed/notarized footer.

## Why It Stands Out

gridOS is not trying to be another plain terminal with an AI form bolted on.
The standout feature is the cockpit: a native multi-pane terminal with a local
visual signal, live system pulse, and an AI Command Helper that treats generated
commands like something to inspect, not blindly execute.

## Install

1. Download `gridOS-1.0.4-12-fe73021.dmg`.
2. Open the DMG.
3. Drag `gridOS.app` into Applications.
4. If you installed an earlier copy, replace it.
5. Launch gridOS from Applications.

If macOS blocks the app, do not bypass Gatekeeper. Report it as a release issue.
