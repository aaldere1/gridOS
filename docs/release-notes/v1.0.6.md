# gridOS 1.0.6

gridOS 1.0.6 is the public-polish release: a sharper icon, cleaner public
screenshots, automatic updates, and a final signed/notarized artifact.

## What's New

- New glass-grid app icon across the macOS icon set.
- README hero rebuilt from the app icon, with product screenshots that avoid
  terminal prompts, local usernames, and process details.
- Sparkle automatic updates are enabled for the direct-download release lane.
- Software Updates settings now expose automatic checks, automatic install, and
  manual Check for Updates.
- Sparkle system profiling is disabled.
- Public privacy docs now describe update checks and GitHub release downloads.
- Sparkle helper binaries are Developer ID signed with secure timestamps inside
  the release artifact.

## Why It Stands Out

The app now feels like something you can confidently send to people: the icon is
distinctive, the README is eye-catching without being misleading, the app has a
real update path, and the direct-download DMG is signed, notarized, stapled, and
Gatekeeper-assessed.

## Install

1. Download `gridOS-1.0.6-14-edda1ee.dmg` from the `v1.0.6` GitHub release.
2. Open the DMG.
3. Drag `gridOS.app` into Applications.
4. If you installed an earlier copy, replace it.
5. Launch gridOS from Applications.

If macOS blocks the app, do not bypass Gatekeeper. Report it as a release issue.
